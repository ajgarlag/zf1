The incubator contains pre-release components for evaluation purposes only.
Components in incubator are undergoing active development.  As changes are
made, there is no guarantee of backward compatibility. 

We welcome feedback based on your evaluation of incubator components.
Your comments are needed to guide the development of these components.

Please make sure to stay current with the latest revisions of incubator
components, so that your comments are most applicable to the current
developments.

Download the files from our Subversion source code repository.
For instructions on how to access our Subversion server,
see http://framework.zend.com/wiki/x/IgE

For more about how to contribute, see: http://framework.zend.com/wiki/x/GgE
